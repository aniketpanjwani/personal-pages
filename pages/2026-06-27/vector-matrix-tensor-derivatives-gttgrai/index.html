Core Habit

When a matrix derivative feels mysterious, reduce it to one scalar output and one scalar input. Write that scalar output as an explicit sum, take the ordinary one-variable partial derivative, then rebuild the pattern.

The main move is: matrix equation $\rightarrow$ one component equation $\rightarrow$ ordinary scalar derivative $\rightarrow$ shape-aware result.

Column-Vector Setup

Suppose $x$ is a $D \times 1$ column vector, $W$ is $C \times D$, and $y$ is a $C \times 1$ column vector:

$$ y = Wx $$

The $i$-th output component is:

$$ y_i = \sum_{j=1}^{D} W_{i,j}x_j $$

So the derivative of one output component with respect to one input component is:

$$ \frac{\partial y_i}{\partial x_j} = W_{i,j} $$

Stacking all those partials gives the Jacobian:

$$ \frac{dy}{dx} = W $$

Row-Vector Setup

CS231n assignment code often stores examples as rows. If $x$ is a $1 \times D$ row vector and $W$ is $D \times C$, then:

$$ y = xW $$

The $j$-th output component is:

$$ y_j = \sum_{k=1}^{D} x_k W_{k,j} $$

Therefore:

$$ \frac{\partial y_j}{\partial x_k} = W_{k,j} $$
Same idea, different orientation. Always check whether examples/features live in rows or columns before deciding where transposes go.

Derivative With Respect To Weights

For the row-vector version $y = xW$, one output component is:

$$ y_j = \sum_{k=1}^{D} x_k W_{k,j} $$

A weight $W_{a,b}$ only affects output component $y_b$. So:

$$ \frac{\partial y_j}{\partial W_{a,b}} = \begin{cases} x_a, & j=b \\ 0, & j \ne b \end{cases} $$

This is why gradients with respect to a weight matrix often look like an outer product: an input activation times an upstream gradient.

Many Examples At Once

Let $X$ be $N \times D$, where each row is one example. Let $W$ be $D \times C$. The score matrix is:

$$ Y = XW $$

One score is:

$$ Y_{i,j} = \sum_{k=1}^{D} X_{i,k}W_{k,j} $$

Changing row $i$ of $X$ only affects row $i$ of $Y$:

$$ \frac{\partial Y_{i,j}}{\partial X_{r,k}} = 0 \quad \text{when } r \ne i $$

For the matching row:

$$ \frac{\partial Y_{i,j}}{\partial X_{i,k}} = W_{k,j} $$

Chain Rule With Matrices

If:

$$ y = VWx $$

define an intermediate:

$$ m = Wx \qquad y = Vm $$

The scalar component form of the chain rule is:

$$ \frac{\partial y_i}{\partial x_j} = \sum_{k} \frac{\partial y_i}{\partial m_k} \frac{\partial m_k}{\partial x_j} $$

Since $\frac{\partial y_i}{\partial m_k}=V_{i,k}$ and $\frac{\partial m_k}{\partial x_j}=W_{k,j}$:

$$ \frac{\partial y_i}{\partial x_j} = \sum_k V_{i,k}W_{k,j} $$

That is exactly the $(i,j)$ entry of $VW$.

Softmax Assignment Connection

In Assignment 1 softmax, your score matrix has the row-example shape:

$$ S = XW $$

The softmax-cross-entropy gradient gives an upstream signal with the same shape as $S$:

$$ dS \in \mathbb{R}^{N \times C} $$

Then the vectorized weight gradient has the familiar matrix product form:

$$ dW = X^T dS $$

Shape check:

$$ X^T: D \times N, \qquad dS: N \times C, \qquad dW: D \times C $$
This is the practical payoff: scalar derivative reasoning explains why the vectorized implementation uses transposes and matrix multiplication.
